## Font usage

1. We use the Inter font for normal text.
2. We use Syne font for headings. Don't ever use the black variant, it's too heavy. Use bold instead. We only use black for when the brand name is written ("Buff").
3. Don't ever use a mono font.

## Communication

1. Don't communicate like a wannabe.
2. Don't use edgy things like "Engine status: High Performance". This doesn't mean anything and is just cringe.
